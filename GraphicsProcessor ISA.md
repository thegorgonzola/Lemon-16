## Processor Instruction Set
 - 00 // No Operation
 - 01 // Store Value to VRAM
 - 02 // Store VRAM to VRAM
 - 03 // Store VRAM to VRAM if Zero
 - 04 // Store VRAM to VRAM if Equal
 - 05 // Store VRAM to VRAM if Less
 - 06 // Store VRAM to VRAM if More
 - 07 // Jump to VRAM
 - 08 // Jump to VRAM if Zero
 - 09 // Return
 - 0A // Add Float
 - 0B // Subtract Float
 - 0C // Multiply Float
 - 0D // Divide Float
 - 0E // Remainder Float
 - 0F // Negative Float
 - 10 // Add Integer
 - 11 // Subtract Integer
 - 12 // Multiply Integer
 - 13 // Divide Integer
 - 14 // Remainder Integer
 - 15 // Negative Integer

# Lemon-16
The Lemon-16 is a small emulator for a virtual 16-bit computer.


## Features
### Processor
 - 16-Bit Processor
 - 16MHz Processor Speed
 - Custom Processor Architecture

### Memory
 - 32 Registers
 - 65KB Stack
 - 2GB RAM (can support up to 4GB)
 - 256MB of VRAM (can support up to 4GB)
 - 16GB SSD (can support up to 9.4 Zettabytes (that's a lot))

### Graphics
 - 1280 x 720 Screen (can support anything you can fit in VRAM)
 - Dedicated Graphics Processor
 - Linear Algebra Functions

# Lemon-16
The Lemon-16 is a small emulator for a virtual 16-bit computer.<br/>

## Features
### Processor
 - 16-Bit Processor
 - 16MHz Processor Speed
 - Custom Processor Architecture (technically a RISC architecture)

### Memory
 - 32 Registers
 - 65KB Stack
 - 32-Bit RAM Addressing (can support up to 4GB; defaults to 2GB)
 - 64-Bit SSD Addressing (can support up to 9ZB; defaults to 16GB)

### Graphics
 - 1280 x 720 Screen (can support anything you can fit in VRAM)
 - Dedicated Graphics Processor
 - Linear Algebra Functions
